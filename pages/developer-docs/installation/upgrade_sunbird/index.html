Sunbird updates are released on a regular basis. For upgrading to the latest version of Sunbird . To update/redeploy sunbird please follow these steps: 1. Update the Sunbird image versions to latest gold version (e.g. PLAYER_VERSION).

  1. Update the configuration as per Sunbird release notes.

  2. Run cd sunbird-devops/deploy

Executing the following command will onboard various APIs and consumer groups.

  1. Run sudo ./deploy-apis.sh /ansible/inventoriesRun

Executing the following command will setup all the sunbird core services.

  1. Run sudo ./deploy-core.sh /ansible/inventories/.

Executing the following command will setup sunbird proxy services.

  1. Run sudo ./deploy-proxy.sh /ansible/inventories/.
